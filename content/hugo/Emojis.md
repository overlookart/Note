## 启用表情符号

在配置文件 `config.toml` 中，将 `enableEmoji` 设置为 `true` 即可启用表情符号。

``` toml
# 启用表情符号
enableEmoji = true
```

## 使用表情符号

在 Markdown 文档中，可以使用 `:emoji:` 语法来插入表情符号。

``` markdown

Hello, :wave:

```

> 效果： Hello, :wave:
