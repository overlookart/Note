| access | 访问，进入 | "ac-" (加强) + "cess"(走) => 进入某处,访问 |

| allocation | 分配，划分 | Object-C 语言里的 alloc => 分配内存 |

| architecture | 架构，建筑 | "arch-" (主要) + "tect"(建筑) => 建筑的主要结构 |
| array | 数组 | Object-C 语言里的 NSArray => 数组 |
| artifact | 工件，制品 | 死记硬背 |

| atomic | 原子的 | Object-C 语言里的 atomic => 原子的 |
